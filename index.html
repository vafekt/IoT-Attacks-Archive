About This Project

Research Context

This hub presents a structured comparative analysis of the available IoT security datasets for training and evaluating machine learning-based intrusion detection systems (IDS). The IoT security landscape is fragmented â€” datasets use different protocols, feature sets, label taxonomies, and testbed configurations.

Our goal is to build a unified AI-based IDS capable of detecting attacks across heterogeneous IoT protocols by harmonizing 16 datasets into a single training corpus, from smart home devices to industrial control systems and building automation.

This site serves as the public-facing reference for the dataset analysis, protocol survey, and attack taxonomy underlying the research project.
